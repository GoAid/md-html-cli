# Syntax Highlight Sample

## C

```c
#include <stdio.h>

int main(){
    printf("Hello World!\n");
    return 0;
}
```

## Ruby

```ruby
print "Hello World!\n"
```

## JavaScript

```js
console.log("Hello World!");
```

## Go

```go
package main

import (
	"fmt"
	"os"
)

func main() {
	fmt.Fprint(os.Stdout, "Hello World!\n")
}
```
